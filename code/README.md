# Documentación

## Entrega

La entrega se realiza vía **aula.usm.cl** en formato `.zip`.

## Multiplicación de matrices

### Programa principal

Primero ejecutar el script de creación de casos de prueba, después ejecutar "make run_all" a la altura del MAKEFILE, esperar a que termine de ejecutarse.

### Scripts
Ambos scripts fueron obtenidos de Geeks For Geeks, con cambios mínimos para ajustarlo al código, cómo el comentar la función "main" en los scripts, créditos al principio del código de cada uno.
Quick sort es con pivote aleatorio y usando la partición de Hoare.

## Ordenamiento de arreglo unidimensional

Algoritmos: MergeSort, QuickSort, std::sort.

### Programa principal
Primero ejecutar el script de creación de casos de prueba, después ejecutar "make run_all" a la altura del MAKEFILE, esperar a que termine de ejecutarse.

### Scripts
Merge Y Quick fueron obtenidos de Geeks For Geeks, con cambios mínimos para ajustarlo al código, cómo el comentar la función "main" en los scripts, créditos al principio del código de cada uno.
Sort usa la función propia de c++, por lo que no se sacó de ningún otro lado.
